# UI Component Guidelines

Use the following libraries for all aesthetic and animation components in this project. Do not introduce custom animation or visual effect implementations when these libraries cover the use case.

## React Bits
- **Docs**: https://react-bits.dev
- General-purpose aesthetic and animation components.

## Aceternity UI
- **Docs**: https://ui.aceternity.com
- Gold standard for high-end landing page components. Use for sophisticated effects like background beams, tracing lines, and sticky scroll reveals.

## Magic UI
- **Docs**: https://magicui.design
- Designed for "design engineers." Use for animated beams, dock menus, shiny borders, and SaaS-style UI elements.

## Motion Primitives
- **Docs**: https://motion-primitives.com
- From the creators of Framer Motion. Use for lower-level, polished animation building blocks like image comparison sliders and layout transitions.
